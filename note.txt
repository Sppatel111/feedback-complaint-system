
Feedback and Complaint Management System
Developed a Flask-based web application to streamline handling of employee feedback and complaints with role-based access for Admins and Users.

Key Features:

User Module: Submit feedback and complaints, track status, and view detailed complaint history.

Admin Module: Manage complaints, respond with attachments, update statuses (Pending/In Progress/Resolved), and manage users.

Search & Filters: Implemented department filter, status filter, and date range filter for complaint records to enable efficient tracking and analytics.

Interactive Dashboard: Built with Bootswatch Lux theme for responsive UI and JavaScript for dynamic filtering.

Database: Designed relational schema using SQLAlchemy with Flask-Migrate for version control.

Tech Stack: Flask, Flask-WTF, Flask-Login, SQLAlchemy, JavaScript, Bootswatch Lux, PostgreSQL/MySQL.
